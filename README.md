## 安装依赖

使用`pnpm`安装依赖：

```
pnpm install
```

## 启动开发服务器

```
pnpm dev
```

## 启动Storybook页面

```
pnpm storybook
```

## @测试

```
pnpm vitest
```

## 查看测试覆盖率

```
pnpm coverage
```
